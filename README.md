# Research Project Tracker – Frontend

## Overview
This is a simple React frontend for a Research Project Tracker system. 
It demonstrates basic UI structure and page components.

## Features
- Login Page UI
- Projects Page UI
- Simple component-based structure

## Technologies Used
- React
- TypeScript

## How to Run
1. Install Node.js
2. Run npm install
3. Run npm start

This project is developed for educational purposes to demonstrate frontend integration concepts.
